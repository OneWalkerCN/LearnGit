# 这是一个git练习笔记

> git是一个免费开源的好工具
> 我们都喜欢git，所以要学习它

## 创建、修改、添加文件

* git init --创建仓库

* git add <文件名> --添加文件到仓库

* git commit -m "<文件的提交细节>" --提交文件

> 为什么有了add为什么要commit文件，因为commit可以提交好多个文件所以可以用add添加多个文件再commit

## 查看文件状态

* git status --查看仓库的状态

* git giff --查看具体修改内容

* git log --查看版本历史记录（--pretty=oneline参数更好的观看）
